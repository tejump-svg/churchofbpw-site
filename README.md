# Church of BPW Website

This repository hosts the source code for the Church of BPW website. It includes all the pages, definitions, arguments, and other content relevant to the church.

## Structure

- **Index Page:** The main landing page for visitors.
- **Canon and Definitions:** Sections defining the core principles.
- **Arguments & Facts:** Pages that lay out philosophical arguments and factual information.

## Getting Started

To run the site locally, clone this repository and open `index.html` in your browser.
